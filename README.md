# Shiyan Teacher PC（实验·教师端 PC 版）

> **本仓库内容为开发中的原型文件，尚未达到可发布版本。**
> 数据结构、界面布局、交互细节均可能继续调整，请勿作为最终产品参考。

幼儿园主题探究活动系统的 **教师端 PC 端（Web）** 原型仓库。整个产品形态为 Web 端 + 手机小程序双端协作，本文仅描述 **PC 端的职责与当前原型状态**。完整设计规范见 [`spec.md`](./spec.md)。

---

## 一、PC 端产品定位

PC 端是教师完成主题探究活动**全流程闭环**的主阵地：

```
事后回忆补录素材 → 整理打标 → 搭建主题网络图 → 撰写月/周/教案 → 提交园所审阅 → 修改通过
```

PC 端承担文档撰写、整理、审阅等需要"较长时间专注操作"的工作；手机端则专注于现场素材的快速采集。

---

## 二、PC 端核心模块

| 模块 | 职责 |
|---|---|
| **首页 / 工作台** | 当前进行中主题概览、待办（待提交 / 被退回 / 新批注）、通知中心 |
| **主题** | 主题列表、新建主题、主题详情（概览 / 资料库 / 网络图 / 月计划 / 周计划 / 教案 / 反思） |
| **主题网络图** | 预设版 + 生成版双轨编辑；主题结束时支持"预设 vs 生成"对照视图 |
| **资料库（主题内）** | 儿童语料（双轴标签：话题对象 + 思维类型）、照片视频、作品图 |
| **月计划** | 1 主题 1 份，嵌入网络图，含周次安排、环境创设、家园共育、资源准备 |
| **周计划** | 5 天 × 一日生活环节结构化表格，单元格可一键创建关联教案 |
| **教案** | 主题相关活动强制关联语料或子探究点；常规 / 节日 / 安全等活动无需关联 |
| **主题反思** | 主题结束时撰写，结合思维类型统计 + 双网络图对照 |
| **月份视图** | 按月聚合本月涉及的主题与文档（只读） |
| **跨主题资料库** | 跨主题检索（按思维类型 / 儿童 / 时间） |
| **文档提交 / 审阅协同** | 草稿 → 提交 → 审阅中 → 通过 / 退回，四态流转 + 行内批注 |

---

## 三、PC 端与手机端的分工

| 终端 | 在主题探究中做的事 | 不做的事 |
|---|---|---|
| **PC 端（Web）** | 撰写月/周/教案、整理语料（打标签、归类）、编辑主题网络图、提交文档、查看园所批注 | （无） |
| **手机端（小程序）** | 上传照片 / 视频素材、查看主题进度与计划、接收审阅与任务通知 | **不提供**文档撰写、标签编辑、网络图编辑 |

素材采集在手机端完成，PC 端做整理与文档化。

### 原型页面

- `teacher-workbench-prototype.html` —— **当前主原型**，是教师工作台 PC 端界面探索的主战场。
- `index.html` —— 早期原型，可能已被主原型取代，保留以备回溯。

### 设计依据

所有原型设计都基于 [`spec.md`](./spec.md) 中的：

- 数据模型（主题 / 语料 / 网络图 / 月/周/教案 / 反思）

- 标签体系（话题对象 + 思维类型双轴）
- 网络图双轨制（预设版 + 生成版）

阅读或评审本仓库前，建议先通读 `spec.md` 以建立完整上下文。
