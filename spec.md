# 幼儿园主题探究活动 APP 设计规范

> 本文档用于指导原型设计，记录 MVP 阶段所有功能模块、数据模型、交互流程和界面信息架构。

---

## 一、产品定位

### 1.1 目标用户

- **主体用户**：幼儿园一线教师（不区分主班 / 配班 / 保育员，班级内部自行分工）
- **审阅用户**：园长 / 教研主任（统称管理员，使用同一管理端）

### 1.2 核心价值

支持教师完成主题探究活动从素材沉淀到文档提交的全流程闭环：

```
事后回忆补录素材 → 整理打标 → 搭建主题网络图 → 撰写月/周/教案 → 提交园所审阅 → 修改通过
```

### 1.3 双端分工

| 终端 | 职责 |
|---|---|
| **Web 端**（教师 / 管理员） | 撰写月计划 / 周计划 / 教案、整理语料库（打标签、归类）、编辑主题网络图、提交文档、园所审阅与批注 |
| **手机端**（教师小程序） | 上传照片 / 视频素材、查看主题进度与计划、接收审阅与任务通知 |

> 手机端在主题探究模块内**不提供文档撰写、标签编辑、网络图编辑**功能。

### 1.4 MVP 范围之外（本 spec 不展开）

家园沟通、研学报名等手机端其他业务不在本 spec 范围内。

## 二、流程框架与数据层级

### 2.1 主题探究活动流程（混合型）

```
园所文化方向（预设）
  └─ 教师选定主题载体（如"泥塑"）
      └─ 抛出话题收集儿童回应 → 形成语料库
          └─ 老师整理语料 → 聚类成子探究点
              └─ 搭建主题网络图（预设版 + 生成版双轨）
                  └─ 月计划（含网络图、周次安排）
                      └─ 周计划（含 5 天 × 一日生活流程表）
                          └─ 教案（主题相关活动按需撰写）
                              └─ 主题反思（主题结束时）
```

### 2.2 数据层级（主题 × 月份双轨并行）

```
学期 (semester)
 ├─ 主题 (theme)            // 主题维度，时长 2-6 周自由设定
 │   ├─ 主题网络图 (theme_map)
 │   │   ├─ 预设版 (preset)
 │   │   └─ 生成版 (emergent)
 │   ├─ 资料库 (resource_pool)
 │   │   ├─ 儿童语料 (utterance)
 │   │   ├─ 照片 (photo)
 │   │   ├─ 视频 (video)
 │   │   └─ 作品图 (artwork)
 │   ├─ 月计划归属 (1 个主题对应 1 份月计划)
 │   ├─ 周计划集合 (多份)
 │   ├─ 教案集合 (多份)
 │   └─ 主题反思 (theme_reflection)
 │
 └─ 月份视图 (month_view)    // 时间维度，聚合视图
     └─ 自动聚合本月涉及的主题、周计划、教案
```

**关键约束：**

- 主题归属于学期，时长由教师定义起止日期，可跨月
- 月份视图不是独立实体，是按时间维度聚合的只读视图（教师提交给园所的月度文档由此生成）
- 周计划、教案直接归属于主题，不直接归属月份
- 资料库挂在主题下，所有素材必须归属某个主题

## 三、核心数据模型

### 3.1 主题 (theme)

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `theme_id` | UUID | 是 | |
| `name` | string | 是 | 主题名称，如"泥塑里的家乡" |
| `class_id` | UUID | 是 | 所属班级 |
| `semester_id` | UUID | 是 | 所属学期 |
| `start_date` | date | 是 | 主题起始日期 |
| `end_date` | date | 是 | 主题结束日期 |
| `origin_description` | richtext | 否 | 主题来源（园所文化方向 + 教师选定理由） |
| `status` | enum | 是 | `planning` / `in_progress` / `finished` |
| `created_by` | UUID | 是 | 创建教师 |
| `created_at` | datetime | 是 | |

### 3.2 儿童语料 (utterance)

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `utterance_id` | UUID | 是 | |
| `theme_id` | UUID | 是 | 所属主题 |
| `content` | text | 是 | 孩子的原话（教师事后录入） |
| `child_name` | string | 否 | 说话儿童（可匿名） |
| `occurred_at` | date | 否 | 发生日期（教师回忆） |
| `scene` | string | 否 | 发生场景（如"集体讨论""区角活动"） |
| `topic_tags` | array<topic_tag_id> | 否 | **话题对象标签**（主题内动态生成，多选） |
| `thinking_tags` | array<thinking_type> | 否 | **思维类型标签**（系统预设枚举，多选） |
| `linked_sub_topic` | UUID | 否 | 关联到主题网络图的某个子探究点 |
| `attachments` | array<media_id> | 否 | 附带的照片 / 录音 / 视频 |
| `created_at` | datetime | 是 | |

#### 3.2.1 话题对象标签 (topic_tag)

- 教师在主题内自由创建，归属于具体主题
- 同一主题内的标签可被多条语料复用
- 跨主题不复用（每个主题的对象维度不同）
- 示例：在"泥塑"主题下可能有"花朵""人物""建筑""动物"等

#### 3.2.2 思维类型 (thinking_type)

系统预设枚举，不可由教师修改：

| 枚举值 | 中文标签 | 说明 |
|---|---|---|
| `observation` | 观察描述 | 孩子对事物的描述性表达 |
| `questioning` | 提出问题 | 孩子主动提问 |
| `causal_reasoning` | 因果推理 | 孩子尝试解释原因或推论结果 |
| `imagination` | 想象联想 | 孩子的想象性表达、类比联想 |
| `transfer` | 经验迁移 | 孩子调用已有经验解释新情境 |

> 思维类型用于主题反思阶段的儿童学习与发展评估，话题对象用于主题前期的子探究点聚类。两轴并存，互不替代。

### 3.3 主题网络图 (theme_map)

每个主题持有两张图：预设版 (preset) 与生成版 (emergent)。

#### 节点 (map_node)

| 字段 | 类型 | 说明 |
|---|---|---|
| `node_id` | UUID | |
| `theme_id` | UUID | |
| `version` | enum | `preset` / `emergent` |
| `parent_node_id` | UUID | 父节点，null 表示中心词 |
| `label` | string | 节点文字 |
| `position` | {x, y} | 在画布上的位置 |
| `linked_utterances` | array<utterance_id> | 该节点关联的支撑语料 |
| `linked_activities` | array<lesson_id> | 该节点关联的教案 |
| `created_at` | datetime | |

**预设版 (preset)：** 主题开始前教师搭建，定稿后较少改动。

**生成版 (emergent)：** 主题进行中根据儿童实际表现持续生长，可由教师从语料聚类"提升"出新节点。

**对照视图：** 主题结束时支持预设版与生成版并排查看，作为"预设与生成"教研反思的依据。

### 3.4 月计划 (monthly_plan)

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `plan_id` | UUID | 是 | |
| `theme_id` | UUID | 是 | 关联主题 |
| `month` | YYYY-MM | 是 | |
| `theme_value_analysis` | richtext | 是 | 主题来源与价值分析 |
| `theme_goals` | richtext | 是 | 主题目标（认知 / 能力 / 情感，或按五大领域） |
| `embedded_theme_map` | reference | 是 | 嵌入当前主题的网络图（指向 theme_map） |
| `weekly_arrangement` | array<row> | 是 | 周次安排表（第几周 / 子主题 / 核心活动） |
| `environment_setup` | richtext | 是 | 环境创设（主题墙、区角材料） |
| `home_collaboration` | richtext | 是 | 家园共育 |
| `resource_preparation` | richtext | 是 | 资源准备 |
| `status` | enum | 是 | 见 3.7 状态机 |
| `created_by` | UUID | 是 | |

### 3.5 周计划 (weekly_plan)

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `plan_id` | UUID | 是 | |
| `theme_id` | UUID | 是 | |
| `week_start_date` | date | 是 | 该周起始日期 |
| `sub_topic` | string | 否 | 本周子主题 |
| `week_goals` | richtext | 是 | 本周目标 |
| `daily_arrangement` | array<cell> | 是 | 5 天 × 一日生活环节表 |
| `area_materials` | richtext | 是 | 区角材料投放（按区角分类） |
| `week_reflection` | richtext | 否 | 一周反思（周末填写） |
| `status` | enum | 是 | 见 3.7 状态机 |

#### 一日生活环节单元格 (daily_cell)

| 字段 | 类型 | 说明 |
|---|---|---|
| `day` | enum | `mon` / `tue` / `wed` / `thu` / `fri` |
| `segment` | enum | `morning_talk`（晨间谈话）/ `group_teaching`（集体教学）/ `area_activity`（区角）/ `outdoor`（户外）/ `meal`（餐点）/ `pre_nap`（午睡前）/ `pre_dismissal`（离园前） |
| `brief` | string | 简要描述（直接显示在表格内） |
| `linked_lesson_id` | UUID | 关联教案（如有，点击单元格跳转） |

### 3.6 教案 (lesson_plan)

不按天必填，仅当涉及主题相关活动（集体教学、特色活动等）时撰写。

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `lesson_id` | UUID | 是 | |
| `theme_id` | UUID | 是 | |
| `weekly_plan_id` | UUID | 否 | 关联周计划（可由周计划表格单元格创建） |
| `title` | string | 是 | 教案名称 |
| `activity_type` | enum | 是 | 见下方枚举，决定"活动来源"是否强制关联 |
| `domains` | array<enum> | 是 | 领域归属（健康 / 语言 / 社会 / 科学 / 艺术，多选） |
| `age_group` | enum | 是 | `nursery`（小班）/ `middle`（中班）/ `senior`（大班） |
| `duration_minutes` | int | 是 | 活动时长 |
| `activity_origin` | richtext | 见 3.6.1 | 活动来源：为什么设计此活动 |
| `linked_utterances` | array<utterance_id> | 见 3.6.1 | 关联的儿童语料 |
| `linked_sub_topics` | array<node_id> | 见 3.6.1 | 关联的主题网络图子节点 |
| `goals` | richtext | 是 | 活动目标 |
| `preparations` | richtext | 是 | 活动准备（经验 / 物质 / 环境） |
| `process` | richtext | 是 | 活动过程（导入 / 展开 / 结束，可分步骤） |
| `extension` | richtext | 否 | 活动延伸 |
| `reflection` | richtext | 否 | 活动反思（活动结束后） |
| `status` | enum | 是 | 见 3.7 状态机 |

#### 3.6.1 活动来源的差异化关联规则

| `activity_type` 枚举值 | 中文 | 是否强制关联语料 / 子探究点 |
|---|---|---|
| `theme_group_activity` | 主题集体活动 | **强制**（必须至少关联 1 条语料或 1 个子探究点） |
| `theme_area_activity` | 主题区角活动 | **强制** |
| `theme_outdoor_activity` | 主题户外活动 | **强制** |
| `routine_activity` | 常规活动 | 非必填 |
| `holiday_activity` | 节日活动 | 非必填 |
| `safety_activity` | 安全教育 | 非必填 |
| `other` | 其他 | 非必填 |

> 强制关联仅在保存为"已提交"状态时校验；草稿状态可暂存。

### 3.7 文档状态机

适用于：月计划、周计划、教案、主题反思。

```
draft（草稿）
  └─[教师点击"提交"]→ submitted（已提交）
        └─[管理员开始审阅]→ reviewing（审阅中）
              ├─[管理员通过]→ approved（通过）
              └─[管理员退回]→ returned（退回修改）
                    └─[教师修改后再次提交]→ submitted
```

**非阻塞性约束：**

- 月计划 `submitted` 后，教师即可开始撰写归属于该主题的周计划和教案，无需等待 `approved`
- 周计划与教案同理
- 文档状态独立流转，互不阻塞

### 3.8 批注 (annotation)

支持行内批注（针对文档具体字段或段落）。

| 字段 | 类型 | 说明 |
|---|---|---|
| `annotation_id` | UUID | |
| `target_doc_type` | enum | `monthly_plan` / `weekly_plan` / `lesson_plan` / `theme_reflection` / `theme_map` |
| `target_doc_id` | UUID | |
| `target_field` | string | 被批注的字段名，如 `theme_goals` |
| `target_range` | {start, end} | 在富文本内的位置范围（如全字段批注则为 null） |
| `content` | text | 批注内容 |
| `author_id` | UUID | 批注人（管理员） |
| `created_at` | datetime | |
| `resolved` | boolean | 教师可标记为已处理 |

### 3.9 主题反思 (theme_reflection)

主题结束时撰写，结构化骨架 + 富文本。

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `reflection_id` | UUID | 是 | |
| `theme_id` | UUID | 是 | |
| `theme_value_review` | richtext | 是 | 主题价值回顾 |
| `goals_achievement` | richtext | 是 | 主题目标达成情况 |
| `children_development` | richtext | 是 | 儿童学习与发展（建议引用思维类型统计与语料证据） |
| `preset_vs_emergent` | richtext | 是 | 预设与生成对照（结合双版本网络图） |
| `improvements` | richtext | 否 | 改进建议 |
| `status` | enum | 是 | 见 3.7 |

## 四、模块清单与信息架构

### 4.1 Web 端教师工作台

```
首页 / 工作台
  ├─ 当前主题概览（进行中的主题卡片，跳转主题详情）
  ├─ 待办（草稿待提交、被退回的文档、园所新批注）
  └─ 通知中心

主题
  ├─ 主题列表（按学期分组，支持筛选状态）
  ├─ 新建主题
  └─ 主题详情
      ├─ 概览（基本信息 + 周期 + 进度）
      ├─ 资料库
      │   ├─ 儿童语料（列表 / 卡片，支持双轴筛选）
      │   ├─ 照片视频（按上传时间 / 子探究点筛选）
      │   └─ 作品图
      ├─ 主题网络图
      │   ├─ 预设版编辑
      │   ├─ 生成版编辑
      │   └─ 对照视图（主题结束时启用）
      ├─ 月计划（1 份）
      ├─ 周计划（多份，按周排序）
      ├─ 教案（多份，按周 / 活动类型筛选）
      └─ 主题反思（1 份，主题结束时撰写）

月份视图
  └─ 按月聚合（展示该月涉及的主题、周计划、教案列表）

资料库（跨主题总库）
  └─ 跨主题检索（按思维类型、按儿童、按时间）

个人设置
```

### 4.2 Web 端管理端

```
首页 / 审阅中心
  ├─ 待审阅列表（按提交时间排序，含文档类型、班级、教师）
  ├─ 审阅中（自己已认领）
  └─ 已通过 / 已退回

班级管理
  ├─ 班级列表
  └─ 班级详情（查看该班所有主题、文档、资料库）

文档审阅页
  ├─ 文档全文（含行内批注高亮）
  ├─ 批注侧栏
  ├─ 操作按钮（通过 / 退回 / 添加批注）
  └─ 历史版本（教师修改后保留前版）

账号与组织
  ├─ 教师账号管理
  ├─ 班级与年级配置
  └─ 模板管理（园所自定义月计划 / 周计划 / 教案模板字段）

主题档案
  └─ 全园主题档案库（可查看任意班级的历史主题）
```

### 4.3 手机小程序（教师端，主题探究模块）

```
首页（主题探究入口卡片）
  └─ 当前主题简要卡片（主题名、进度、本周关键信息）

主题详情（只读视图）
  ├─ 概览
  ├─ 网络图（只读，可缩放查看）
  ├─ 本周计划（只读）
  └─ 资料库浏览

上传素材
  ├─ 选择主题（系统智能预判，可调整）
  │   预判规则：依据当前日期匹配进行中的主题；如多个则按周计划归属判定
  ├─ 选择类型（照片 / 视频）
  ├─ 从相册选择 / 拍摄
  └─ 提交（异步上传）

通知中心
  ├─ 园所审阅反馈（被退回、被通过、新批注）
  └─ 任务提醒（如"本周周计划尚未提交"）

个人中心
```

## 五、关键交互流程

### 5.1 主题创建 → 文档提交完整流程

```
1. 教师 Web 端"新建主题"
   └─ 填写主题名、起止日期、来源描述

2. 进入主题详情 → 主题网络图（预设版）
   └─ 拖拽节点搭建预设网络图

3. 主题进行中：手机端上传照片视频 → 自动归属该主题
   Web 端：教师事后录入儿童语料 → 打话题对象标签（建议补思维类型标签）

4. 教师在资料库中查看语料聚类
   └─ 选中若干语料 → 右键"提升为子探究点" → 自动在生成版网络图加节点

5. 教师撰写月计划
   ├─ 系统自动嵌入主题网络图
   ├─ 周次安排表关联主题周期
   └─ 保存为草稿 → 提交

6. 教师撰写周计划（无需等月计划通过）
   ├─ 5 天 × 一日生活环节表
   └─ 在表格单元格中点击"+"创建教案

7. 教师撰写教案
   ├─ 选择活动类型
   ├─ 若为主题相关活动，必须关联语料或子探究点
   └─ 提交

8. 管理员审阅
   ├─ 工作台看到"待审阅"
   ├─ 进入文档审阅页
   ├─ 在字段或段落上添加批注
   └─ 通过 或 退回

9. 教师收到通知（Web + 手机）
   ├─ 通过：完成
   └─ 退回：查看批注 → 修改 → 重新提交

10. 主题结束
    ├─ 教师切换网络图至"对照视图"
    ├─ 撰写主题反思（引用思维类型统计、子探究点完成情况）
    └─ 提交反思
```

### 5.2 儿童语料整理流程

```
教师 Web 端进入主题 → 资料库 → 儿童语料

录入：
  点击"新增语料"
  ├─ 输入孩子原话
  ├─ 选择说话儿童（可匿名）
  ├─ 选发生场景与日期（可选）
  ├─ 打话题对象标签
  │   ├─ 选择已有标签 或
  │   └─ 新建标签（仅在当前主题内生效）
  ├─ 打思维类型标签（系统预设五选多）
  └─ 保存

聚类与提升：
  列表视图支持按话题对象筛选 / 按思维类型筛选 / 按儿童筛选
  多选语料 → "提升为子探究点" → 自动在生成版网络图添加节点 → 节点自动关联这些语料

证据反查：
  在主题反思中引用"思维类型统计" → 点击某一类 → 查看该类的所有语料
```

### 5.3 行内批注交互

```
管理员在文档审阅页：
  ├─ 选中富文本中的某段文字 → 浮出"添加批注"按钮
  ├─ 输入批注内容 → 提交
  └─ 该段文字高亮，右侧批注栏出现条目

教师查看：
  ├─ 进入文档（被退回状态）
  ├─ 高亮段落点击 → 批注侧栏定位
  ├─ 修改后点击"标记已处理"
  └─ 全部处理完后重新提交
```

## 八、设计约束与备注

### 8.1 智能预判逻辑

**手机端上传素材时的主题预判：**

```
输入：当前日期 + 当前用户所属班级
查询：该班级在当前日期处于 in_progress 状态的主题
  ├─ 命中 1 个：默认归属
  ├─ 命中多个：按优先级（开始日期最近）默认归属，提示用户可调整
  └─ 命中 0 个：弹窗提示"当前无进行中的主题"，教师手动选择
```

### 8.2 富文本编辑器

月计划 / 周计划 / 教案 / 反思 / 语料的富文本字段建议统一一套编辑器组件，支持：

- 基本格式（粗体 / 列表 / 标题）
- 插入图片（从主题资料库选）
- 插入语料引用（从主题语料库选，显示为可点击的引用块）
- 插入子探究点引用（从主题网络图选）

### 8.3 文件存储

- 照片：单张建议压缩至 1MB 以内
- 视频：单条建议 100MB 以内，超出提示压缩
- 文档导出：月计划 / 周计划 / 教案 / 反思支持导出 PDF / Word（提交园所外的正式存档）

### 8.4 命名约定

- 数据库字段：snake_case（如 `theme_id`、`linked_utterances`）
- 枚举值：snake_case（如 `in_progress`、`theme_group_activity`）
- 前端组件：PascalCase
- 状态名 / 类型名：在 UI 中以中文呈现，代码中保留英文
